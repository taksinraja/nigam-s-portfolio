# Nigam Bhujel Portfolio

A responsive portfolio website for a fashion model. Includes a hero section, about section, works carousel, services, and contact information.

## Features

- Responsive layout (desktop, tablet, mobile)
- Hero section with overlay and CTA links
- About section with grid layout
- Portfolio section with horizontal scroll, next/prev buttons, slide indicator, and progress bar
- Services section with responsive behavior
- Footer with contact CTA and social links

## Tech Stack

- HTML5
- CSS3
- JavaScript (vanilla)

## Installation

1. Clone repository:
   ```bash
   git clone https://github.com/taksinraja/nigam-s-portfolio.git
   cd nigam-s-portfolio
   ```
2. Open `index.html` in your browser.

## Usage

- Click `PREV`/`NEXT` to navigate portfolio slides and watch the progress bar update.
- Explore responsive breakpoints by resizing the browser.

## Deployment

This project can be deployed with GitHub Pages, Netlify, or Vercel.
